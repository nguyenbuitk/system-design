Flow hoạt động:

  1. User đặt hàng → Business Service lưu vào DB (status: PENDING)
  2. Business Service gửi order vào queue "order_processing"
  3. API trả response ngay: "Order received, processing in background"
  4. Order Worker xử lý order từ queue (validate, email, update status)
  5. Order status được update: PENDING → COMPLETED
