So sánh: Synchronous vs Asynchronous

Xử lý đồng bộ (Sync) Xử lý bất đồng bộ (Async với Queue)
User phải đợi 2-3 giây User nhận response ngay (0.7s)
Browser bị block Browser không bị block
Email service chậm → toàn bộ API chậm Email xử lý background, không ảnh hưởng API
Không thể scale workers Có thể scale workers để xử lý peak load
Nếu service down → không thể đặt hàng Nếu worker down → orders vẫn được lưu, xử lý sau
